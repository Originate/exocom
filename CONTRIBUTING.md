# ExoCommMock-JS Developer Guidelines

## Install

* `npm i`
* add `./bin/` to your PATH


## Development

* run `watch` in a separate terminal to auto-compile changes


## Testing

* run all tests: `spec`
* run unit tests: `tests`


## Update

* `david`


## Deploy a new version

```
npm run update
npm version <patch|minor|major>
npm publish
```
